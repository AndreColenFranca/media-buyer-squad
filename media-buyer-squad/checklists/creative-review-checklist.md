# Creative Review Checklist

## Pre-Production Review

### Strategy Alignment

- [ ] Creative aligns with campaign objective
- [ ] Hook addresses target audience pain point
- [ ] Angle matches audience awareness level
- [ ] Offer is clear and compelling
- [ ] CTA drives desired action

### Brief Completeness

- [ ] Creative brief is complete
- [ ] Target audience defined
- [ ] Key messaging documented
- [ ] Visual requirements specified
- [ ] Timeline agreed upon

---

## Creative Elements Review

### Hook (First 3 Seconds)

- [ ] **Pattern Interrupt**: Stops scroll immediately
- [ ] **Relevance**: Speaks directly to target audience
- [ ] **Clarity**: Message is clear and specific
- [ ] **Intrigue**: Creates curiosity or urgency
- [ ] **Visual Impact**: First frame is compelling

**Hook Rate Target**: > 15%

**Hook Types to Test**:

- [ ] Pattern Interrupt
- [ ] Curiosity Gap
- [ ] Social Proof
- [ ] Problem Agitation
- [ ] Direct Benefit

---

### Body (3-20 Seconds)

#### Problem/Agitation

- [ ] Problem is specific and relatable
- [ ] Pain point is clearly articulated
- [ ] Audience sees themselves in the problem
- [ ] Emotional connection established

#### Solution

- [ ] Solution is clearly presented
- [ ] Unique mechanism or benefit highlighted
- [ ] Transformation or outcome shown
- [ ] Social proof or credibility included

#### Proof

- [ ] Results or testimonials shown
- [ ] Numbers/stats are specific
- [ ] Before/after demonstrated (if applicable)
- [ ] Authority or expertise established

---

### Call-to-Action (Last 3-5 Seconds)

- [ ] CTA is clear and specific
- [ ] Action is easy to understand
- [ ] Benefit of clicking is obvious
- [ ] Visual cue (arrow, button) included
- [ ] Urgency or scarcity (if applicable)

**CTA Examples**:

- [ ] "Click below to [benefit]"
- [ ] "Tap the link to [benefit]"
- [ ] "Get [thing] now"
- [ ] "Learn more here"
- [ ] "Claim your [offer]"

---

## Copy Review

### Primary Text (125-150 words)

- [ ] Hook is in first line
- [ ] Problem → Solution flow
- [ ] Benefits > Features
- [ ] Social proof included
- [ ] CTA is clear
- [ ] Length is 125-150 words
- [ ] Line breaks for mobile readability
- [ ] No jargon or complex language
- [ ] Conversational tone
- [ ] Proofread (no typos)

**Character Count**: **\_** (max: 2200)

### Headline (40 characters max)

- [ ] Reinforces main benefit
- [ ] Compelling and clear
- [ ] 40 characters or less
- [ ] No generic phrases
- [ ] Tested variations

**Character Count**: **\_** (max: 40)

### Description (30 characters max)

- [ ] Secondary benefit or detail
- [ ] 30 characters or less
- [ ] Adds value to headline
- [ ] Not redundant

**Character Count**: **\_** (max: 30)

---

## Visual/Video Review

### Technical Specifications

- [ ] **Format**: 9:16 (1080×1920) for Stories/Reels
- [ ] **Format**: 1:1 (1080×1080) for Feed
- [ ] **Format**: 16:9 (1920×1080) for desktop (optional)
- [ ] **Length**: 15-30 seconds
- [ ] **File Type**: MP4, H.264
- [ ] **File Size**: < 4GB
- [ ] **Resolution**: 1080p minimum
- [ ] **Frame Rate**: 30fps minimum

### Video Quality

- [ ] Clear audio (no background noise)
- [ ] Good lighting (face is visible)
- [ ] Stable footage (not shaky)
- [ ] In focus (not blurry)
- [ ] Professional editing
- [ ] Smooth transitions
- [ ] Color grading (if applicable)

### Captions/Text

- [ ] **Captions**: Burned-in (not auto-captions)
- [ ] **Font**: Large, readable on mobile
- [ ] **Color**: High contrast (white on dark, black on light)
- [ ] **Position**: Bottom third (not covering face)
- [ ] **Accuracy**: Matches spoken words
- [ ] **Timing**: Synced with audio
- [ ] **Spelling**: No typos

### Mobile Optimization

- [ ] Looks good on mobile (test on phone)
- [ ] Text is readable on small screen
- [ ] Important elements in safe zone
- [ ] Works without sound (captions)
- [ ] First frame is compelling (thumbnail)

---

## Brand Compliance

### Brand Guidelines

- [ ] Logo usage follows guidelines
- [ ] Colors match brand palette
- [ ] Fonts match brand fonts
- [ ] Tone matches brand voice
- [ ] Messaging aligns with brand values

### Legal/Compliance

- [ ] No prohibited claims
- [ ] Disclaimers included (if required)
- [ ] Copyright cleared (music, footage)
- [ ] Testimonials are genuine
- [ ] Before/after photos are real

---

## Platform Compliance (Meta)

### Ad Policies

- [ ] No prohibited content
- [ ] No misleading claims
- [ ] Text in image < 20% (if applicable)
- [ ] Landing page matches ad
- [ ] No sensational language
- [ ] No personal attributes targeting language
- [ ] Age-appropriate content

### Common Rejections to Avoid

- [ ] "You" language (use sparingly)
- [ ] Exaggerated claims ("Lose 50 lbs in 1 week")
- [ ] Before/after without disclaimer
- [ ] Shocking/sensational content
- [ ] Too much text in thumbnail

---

## Testing Readiness

### Variations

- [ ] 3-6 hook variations created
- [ ] 2-3 angle variations per hook
- [ ] Total: 6-18 ad variations
- [ ] Each variation has unique element
- [ ] Testing matrix documented

### Constants vs Variables

- [ ] Only ONE variable changing per test
- [ ] Constants clearly documented
- [ ] Variables clearly documented
- [ ] Testing hypothesis defined

**Testing**:

- [ ] Phase 1: Hook testing (6 variations)
- [ ] Phase 2: Angle testing (3 variations)
- [ ] Phase 3: Visual testing (3 variations)

---

## Performance Benchmarks

### Expected Metrics

- [ ] Hook Rate: > 15%
- [ ] CTR: > 1.5%
- [ ] CVR: > 1.0%
- [ ] CPA: < Target
- [ ] ROAS: > 3.0 (e-commerce)

### Success Criteria Defined

- [ ] Minimum impressions: 1000
- [ ] Minimum clicks: 500
- [ ] Minimum conversions: 10
- [ ] Minimum days: 3
- [ ] Kill criteria documented
- [ ] Scale criteria documented

## Common Mistakes to Avoid

### Hook Mistakes

- ❌ Generic opening ("Are you ready to...")
- ❌ Slow start (no pattern interrupt)
- ❌ Unclear message
- ❌ Not mobile-optimized
- ❌ Boring first frame

### Body Mistakes

- ❌ Too long (> 30 seconds)
- ❌ No clear problem/solution
- ❌ Features instead of benefits
- ❌ No social proof
- ❌ Poor pacing

### CTA Mistakes

- ❌ No clear CTA
- ❌ Weak CTA ("Click here")
- ❌ No visual cue
- ❌ CTA too early or too late
- ❌ Multiple CTAs (confusing)

### Technical Mistakes

- ❌ Wrong aspect ratio
- ❌ Low resolution
- ❌ No captions
- ❌ Poor audio quality
- ❌ File too large
