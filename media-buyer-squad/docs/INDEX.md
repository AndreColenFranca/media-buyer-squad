## 🚀 Quick Start Commands

### Ativar Agentes

```bash
@ad-midas              # Estratégia e liderança
@performance-analyst   # Métricas e otimização
@creative-analyst      # Hooks e copy
@pixel-specialist      # Tracking e pixel
@media-buyer          # Execução e consulta
```

### Usar Skills

```bash
@ad-midas *structure-campaign          # Estruturar campanha
@performance-analyst *diagnose-metrics # Diagnosticar métricas
@creative-analyst *generate-hooks      # Gerar hooks
@pixel-specialist *audit-tracking      # Auditar tracking
```

### Executar Scripts

```bash
node scripts/diagnose-metrics.js       # Analisar performance
node scripts/generate-hooks.js         # Gerar hooks
node scripts/audit-tracking.js         # Auditar pixel
node scripts/evaluate-kill-scale.js    # Decisão kill/scale
node scripts/monitor-campaigns.js      # Monitorar 24/7
```
