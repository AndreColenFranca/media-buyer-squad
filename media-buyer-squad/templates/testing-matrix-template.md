## Decision Criteria

### Kill Criteria (Pause Ad)

- [ ] Hook rate < 10% after 1000 impressions
- [ ] CTR < 1.0% after 500 clicks
- [ ] CPA > Target × 2.0
- [ ] No conversions after $500 spend
- [ ] 3 days of data collected

### Keep Criteria (Continue Testing)

- [ ] Hook rate 10-15%
- [ ] CTR 1.0-1.5%
- [ ] CPA within Target × 1.5
- [ ] Some conversions
- [ ] Need more data (< 3 days)

### Scale Criteria (Increase Budget)

- [ ] Hook rate > 15%
- [ ] CTR > 1.5%
- [ ] CPA < Target × 0.8
- [ ] Minimum 10 conversions
- [ ] Frequency < 3.0

## Testing Best Practices

### DO:

✅ Test ONE variable at a time (constants vs variables)  
✅ Give each test minimum 3 days  
✅ Collect minimum 1000 impressions per variation  
✅ Document all results  
✅ Kill bottom 50% after each phase  
✅ Scale top performers  
✅ Build creative library from winners  
✅ Refresh creative every 2-4 weeks

### DON'T:

❌ Test multiple variables simultaneously  
❌ Kill ads before 3 days  
❌ Make decisions on insufficient data  
❌ Forget to document learnings  
❌ Stop testing after finding one winner  
❌ Let creative run until fatigue  
❌ Ignore frequency metrics  
❌ Skip phases (test systematically)
