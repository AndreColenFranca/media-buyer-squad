## Copy Frameworks

### Framework 1: PAS (Problem-Agitate-Solution)

```
[Hook: Identify problem]

[Agitate: Make problem worse]

[Agitate more: Show consequences]

[Solution: Introduce your offer]

[Proof: Social proof or results]

[CTA: Drive action]
```

**Best for**: Cold traffic, problem-aware audience

---

### Framework 2: AIDA (Attention-Interest-Desire-Action)

```
[Attention: Grab attention with hook]

[Interest: Build interest with unique angle]

[Interest: Show what makes you different]

[Desire: Create desire with benefits/results]

[Desire: Add social proof]

[Action: Clear CTA]
```

**Best for**: Warm traffic, solution-aware audience

---

### Framework 3: BAB (Before-After-Bridge)

```
[Before: Describe current state]

[After: Paint picture of desired state]

[Bridge: Show how your solution gets them there]

[Proof: Add credibility]

[CTA: Drive action]
```

**Best for**: Transformation-focused offers

---

### Framework 4: STORY

```
[Setup: Set the scene - where you/they were]

[Struggle: Describe the challenges faced]

[Discovery: What changed everything]

[Transformation: The results achieved]

[Offer: How they can get same results]

[CTA: Drive action]
```

**Best for**: Authority-building, warm traffic

## Copy Checklist

### Before Writing

- [ ] Know your audience (demographics, pain points, desires)
- [ ] Know your offer (product, price, guarantee)
- [ ] Know your angle (mechanism, transformation, authority)
- [ ] Choose your framework (PAS, AIDA, BAB, STORY)

### While Writing

- [ ] Hook stops scroll (first 3 words are critical)
- [ ] Problem is specific and relatable
- [ ] Solution is clear and compelling
- [ ] Benefits > Features
- [ ] Social proof included
- [ ] CTA is action-oriented
- [ ] Length is 125-150 words

### After Writing

- [ ] Read out loud (does it flow?)
- [ ] Check word count (125-150 ideal)
- [ ] Check character limits (Primary: 2200, Headline: 40, Description: 30)
- [ ] Remove jargon and complex words
- [ ] Add line breaks for readability
- [ ] Test with target audience if possible

## Copy Best Practices

### DO:

✅ Start with a strong hook (first 3 words matter most)  
✅ Use short sentences and paragraphs  
✅ Add line breaks for mobile readability  
✅ Use specific numbers ("500+ brands" not "many brands")  
✅ Include social proof  
✅ Make CTA clear and action-oriented  
✅ Write like you talk (conversational tone)  
✅ Focus on benefits, not features  
✅ Address objections  
✅ Create urgency (if genuine)

### DON'T:

❌ Use generic hooks ("Are you ready to...")  
❌ Write long paragraphs (max 2-3 lines)  
❌ Use jargon or complex language  
❌ Make unsubstantiated claims  
❌ Forget the CTA  
❌ Be too salesy or pushy  
❌ Use all caps or excessive punctuation!!!  
❌ Ignore mobile formatting  
❌ Copy competitors exactly  
❌ Forget to proofread

## Hook Library

### Problem-Focused Hooks

- "Struggling to [problem]?"
- "If you're still [mistake], you're [consequence]..."
- "Your [metric] is [symptom]? Here's why..."
- "The reason you can't [goal] is [reason]..."
- "Still [old way]? That's why you're [consequence]..."

### Curiosity Hooks

- "What [authority] doesn't tell you about [topic]..."
- "The #1 [mistake/reason] that's [consequence]..."
- "The secret that [social proof] use to [outcome]..."
- "I spent $[amount] on [topic] so you don't have to..."
- "[Number] things nobody tells you about [topic]..."

### Social Proof Hooks

- "How [we/I] went from [before] to [after] in [timeframe]..."
- "The exact system [number]+ [people] use to [outcome]..."
- "From [before] to [after] with this [method]..."
- "[Authority/Name] used this to [achievement]..."
- "How [number] [people] achieved [outcome]..."

### Pattern Interrupt Hooks

- "Stop [action] if you're [pain point]..."
- "This [descriptor] but it works..."
- "[Unexpected statement] and here's why..."
- "I'm about to show you something [descriptor]..."
- "If you're still [mistake], you're [consequence]..."

### Direct Benefit Hooks

- "[Outcome] in [timeframe] without [objection]..."
- "The fastest way to [outcome] (even if [objection])..."
- "Get [outcome] without [objection 1] or [objection 2]..."
- "How to [outcome] in [timeframe] starting [when]..."
- "[Outcome] guaranteed in [timeframe] or [guarantee]..."
